Markabb

All Ruby BBCode parser for Ruby and Rails.

Install

To install Markabb add it to your gemfile

gem 'markabb'

and run a bundle install to get it.

If you want to use Markabb without bundler run gem install markabb and add require 'markabb' to your code.

Usage

Configuring

Before you can use Markabb you need to configure it, to do this call Markabb.configure and pass a block to it. To use the default config pass an empty block.

Markabb.configure do |c|
end

The most common config is disable_TAGNAME which allows you to disable a specific tag or group, for example to disable the bold tag you would use c.disable_bold = true.

Parsing

To parse a string simply pass it to Markabb.parse, the return value will be the parsed HTML. You can pass config to parse in a block like this:

Markabb.parse "[b]Bold Text[/b]" do |c|
c.disable_bold = true
end
